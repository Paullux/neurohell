# NeuroHell

NeuroHell est un FPS doom-like développé avec Three.js, combinant un univers infernal, une narration immersive et une direction artistique inspirée de l'industriel, du metal et de la science-fiction.

Le projet mélange gameplay rapide, narration sombre et expérimentation technique autour du web, du rendu 3D et de l'IA.

---

## 🚀 Lancer le projet

Clone le repo :

```bash
git clone https://github.com/Paullux/neurohell.git
cd neurohell
```

Puis lance un serveur local (exemple avec Python) :

```bash
python -m http.server 8080
```

Ensuite ouvre :

```
http://localhost:8080
```

---

## 🎮 Fonctionnalités

* FPS en temps réel avec Three.js
* Système d’armes (plasma, sniper, gauntlet)
* HUD complet (santé, énergie, armes, minimap)
* Système de démons avec animations et IA simple
* Musique dynamique (combat / idle / mort)
* Level design avec collisions (Octree)
* Support HDRI pour ambiance infernale

## 🧠 Vision du projet

NeuroHell raconte l’histoire d’un ancien militaire du XXIe siècle plongé en enfer, cherchant à remonter vers le purgatoire pour sauver son âme et retrouver celle qu’il aime.

Le monde mélange :

* technologies futuristes
* entités démoniaques anciennes
* espace intemporel
